=======================
Developer Documentation
=======================

The best way to explore the :doc:`api` is to use the ``--debug`` ndcli
flag. Enabling it will print both the request and the response in JSON (without
the JSON-RPC fields).

Before you implement your own client, you should check if :doc:`dimclient` meets
your needs.
